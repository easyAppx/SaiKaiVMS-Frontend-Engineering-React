# Getting Started Contributing to Sakai-VMS-Frontend-Engineering

 1. ### `git clone https://github.com/global-pan-africanism-network/Sakai-VMS-Frontend-Engineering.git`
 2. ### `cd Sakai-VMS-Frontend-Engineering`
 3. ### `npm install`
 
 4. ### `git checkout -b your-branch-name`

 5. ### `npm start`

## if You Already have a codebase without an Upstream branch

In the project directory, you can run:

### `git init`
### `git add .` for staging
### `git commit -m "commit-message"` 
### `git remote add https://github.com/global-pan-africanism-network/Sakai-VMS-Frontend-Engineering.git` adding a new upstream branch
### `git fetch github` for fetching latest changes in the Remote Branch
### `git checkout -b your-branch-name` for Creating Your Development Branch 
### `git push -u origin <Your-branch-Name>`

if you Face Any error feel free to ask for help...☺️
